# Agents

This folder defines stable operating rules for agents.

Put here:
- how agents take work
- when to update Jira
- when to create Story, Bug, Spike, or Subtask
- branch and cleanup expectations

Do not put here:
- provider-specific tool syntax
- project-specific domain rules

Read first:
- `operating-model.md`
- `story-lifecycle.md`
- `branch-hygiene.md`
